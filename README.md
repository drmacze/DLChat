# DLChat (Dlavie Chat)

A production-ready fullstack mobile chat + social app built with Expo, Express, PostgreSQL, Socket.IO.

## Stack
- Frontend: Expo / React Native (iOS + Android)
- Backend: Express 5 + PostgreSQL + Drizzle ORM
- Realtime: Socket.IO
- Auth: 
- Storage: Supabase
- Push Notifications: Expo Push API

## Build
```bash
eas build --platform android --profile preview  # APK
eas build --platform ios --profile preview      # IPA
```
